# gotools（golang工具包）

## 1. 安装与更新

安装：`go get github.com/asktop/gotools`  
更新：`go get -u github.com/asktop/gotools`

## 2. 模块
| 模块 | 说明 |
| --- | --- |
|afile|文件工具|
|amath|数学工具（decimal.Big）|
|asort|slice切片、map排序工具|
|astring|string字符串工具|
|atime|时间工具|
|cast|interface数据类型转换|
|gzip|gzip压缩工具|
|omap|有序map|
|secret|base64、md5、sha1等加密工具|
|slice|切片工具|
|unique|唯一字符串工具|
